# 版本号2.2.2

## 更改说明

新增了几个接口，分别是

--------------------------------------------------------------------------------------------------------------------

ori_data-> (射频直接接受的原始数据)，解调后的数据(fsk_data_out)复用一个端口

order_in，order_out命令端口

--------------------------------------------------------------

下面是端口分配情况

fsk_data_out用<u>**C2H1_0**</u>，解调后的数据送到Tx2

ori_data用**C2H1_1**，AD9361直接接受的数据

fsk_data_in用**H2C0**，TX2给FPGA的数据

**C2H0**是命令端口 **H2C1**是命令端口



# 版本号2.2.1
